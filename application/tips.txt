==========IMPORTANT===============

Pinul LCD_RD de la display trebuie neaparat conectat la RPI (pe Arduino la A0)!!!!
Asta ca sa mearga touchscreen-ul, altfel valorile sunt gresite

==========IMPORTANT===============




*Change screen resolution:

set in /boot/config.txt:

framebuffer_width=320
framebuffer_height=240


*Read all gpios from command line:

gpio allreadall


Remote debugging:

start the server on the target 
sudo gdbserver 192.168.2.201:1995 /home/pi/licenta/application/build/application &

and then attach from host:

gdb-multiarch application
where application is the ARM compiled version of the app

run command in gdb:
target remote 192.168.2.201:1995
